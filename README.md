# Music Application
## 音乐 播放器 应用
### 功能： 
包括加载音乐列表，播放音乐，暂停音乐，播放下一首音乐，播放上一首音乐，调整音乐进度
### 使用： 
1. 引入依赖包 
2. 配置 
3. 编译运行
### 安装： 
1. 进入 app/release 目录 
2. **安装** release 目录下的 app-release.apk 文件
### 首页： 
1. 初始化加载音乐列表 
2. 上拉刷新从系统的Media数据库中加载更多音乐
